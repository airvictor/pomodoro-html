# Pomodoro 计时器

一款极简风格的 Pomodoro 计时器，采用精美的七段数码管显示、天气小组件，并支持自定义专注/休息时长。

## 功能特点

- **多种计时模式**：60 分钟、40 分钟（默认）、15 分钟和 5 分钟预设
- **自定义时长**：可直接调整计时器时长（1-120 分钟）
- **七段数码管时钟**：实时显示当前时间的经典 LED 风格数字时钟
- **会话追踪**：进度圆点直观展示已完成的专注会话
- **天气小组件**：自动获取位置并显示当前温度
- **声音提醒**：计时结束时播放提示音（叮咚、铃声、数字音效）
- **深色/浅色主题**：一键切换，偏好设置自动保存
- **设置持久化**：所有偏好保存至 localStorage

## 使用方法

直接在现代浏览器中打开 `pomodoro.html`，无需服务器或构建步骤。

### 计时器控制

- **开始/暂停**：点击主按钮启动或暂停计时器
- **重置**：点击重置按钮重新开始当前会话
- **模式选择**：在 60m、40m、15m、5m 预设中切换
- **快速调整**：使用 +/− 按钮或直接输入自定义时长

### 设置面板

点击齿轮图标可以：
- 自定义各模式时长（1-120 分钟）
- 开关声音提醒
- 选择提示音效（叮咚、铃声、数字音效）
- 切换深色模式

## 工作原理

Pomodoro  Technique 采用专注工作时段（通常 25-30 分钟）配合短休息的方式。每完成 4 个专注时段后，系统自动触发一个长休息。

```
专注时段 → 短休息 → 专注时段 → 短休息 → 专注时段 → 短休息 → 专注时段 → 长休息
```
